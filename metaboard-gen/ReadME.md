Online ArUco markers generator
====

### Usage

>要搭建本地服务器运行，先安装 nodejs
>
>1. 使用http-server
>
>   ```node
>   # 全局安装
>   npm install -g http-server
>   # 在 index.html 目录下打开命令行并运行 http-server
>   http-server
>   ```
>
>2. vscode中使用 live server
>
>   安装 live server 插件
>
>   安装完毕后 vscode 右下角会有 Go Live 按钮 点击即可在工作文件夹开启本地服务器
